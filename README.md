# Housing Price Prediction

Regression analysis predicting housing sale prices using the **Ames, Iowa housing dataset** (2,000 homes with 80+ features).

## What I did
- Cleaned and preprocessed the dataset (handled missing values, outliers)
- Built multiple regression models to predict sale price
- Evaluated model performance using cross-validation

## Tools
R, `caret`, `readr`, `ISLR`

## Dataset
`ames2000_NAfix.csv` — Ames housing data with NA values pre-fixed
